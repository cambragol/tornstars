Middle States Keybinds

Epic is mostly menu driven. There are a few keyboard shortcuts to start menu's or speed access during combat.

Ship Computer Menu

This menu will open when your ship docks to a station.

You can also use the "o" (oh) key to open or close the Ship Computer Menu. From here you can access Navigation, Trade, or Save game menu items.

Start Communication

Target another ship and press the "CTRL-c" (control key plus C) to attempt communications. If the target ship supports communications, it will start a dialog. Most ships that support communications will begin cyan in color. After the first communication attempt they will revert to the standard faction standing color.

Hack Cargo Pods

You can convert pirated cargo pods to your faction by pressing the "j" (jay) key. This will convert all lose floating cargo within 20km to your faction. Once converted you can load and unload the pods quickly.

Overview

Open Ship Computer Menu = o

Establish Ship to Ship Communications = Ctrl-c

Convert Pirated Cargo Pods = j
