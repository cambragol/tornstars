Save Game

Press this button to save your game. This save button is only available when you are at least 1000km from any station or Lpoint.

You can load your game by pressing escape to leave the main space flight screen. You should then see the options menue including a load game option. When you load you will return to the location you saved. Your wingmen will return to thier save locations as well.

You can also save from inside the station. Dock to a non-hostile station and check this menue again.
