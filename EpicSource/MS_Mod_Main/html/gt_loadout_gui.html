Ship Fitter Screen

This GUI screen allows the player to view all ships systems and weapons, as well as to replace, remove, repair, or rearm them.

In addition it displays systems and weapons related information such as:

- Heat Sink Rate
- Heat Build Rate
- Reactor Power
- Power Draw

Replacing, removing and repairing systems can only be done while docked in a shipbay.

Rearming systems can be performed when within close proximity to a station.

Ship systems can be viewed at anytime.
