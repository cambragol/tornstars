Player Fleet Status Screen

This GUI screen displays a list of the player ship and all player wingmen.

In addition it displays important ship specific information such as:

- Ship Class
- Hull Integrity
- Fuel Remaining
- Current System
- Wingman Group
- Daily Upkeep Cost
- Fuel per Day
- Approximate Fuel per Jump
- Maximum Fuel
- Carried Cargo

Finally, it displays limited information about ship's systems.
