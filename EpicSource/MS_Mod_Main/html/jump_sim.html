Jump Simulator

This GUI lets you select an fly a simulated capsule jump experience.

Open the GUI. Select a ship. Select a Route. Press Start.

You will be transported into the ship and started on a course of waypoint.

Fly through each waypoint in turn. If you pass within 1km of a waypoint, it will give a valid beep.

If you miss a waypoint, it will give an invalid beep. Try for fastest time.
